# 🌙 Luna - The Mystical Wanderer

A beautiful character website featuring Luna, the mystical wanderer, with an animated galaxy background.

## 🚀 Live Website

Your website will be available at: **https://mr6andrew.github.io/Luna-website/**

## 🛠️ Setup Instructions

### To enable GitHub Pages:

1. Go to your repository: https://github.com/mr6andrew/Luna-website
2. Click on **Settings** tab
3. Scroll down to **Pages** section (in the left sidebar)
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch
6. Click **Save**

Your website will be live in a few minutes at: **https://mr6andrew.github.io/Luna-website/**

## ✨ Features

- `index-galaxy.html` - Main website with galaxy animation

- `styles.css` - Main styling
- `script.js` - Interactive functionality

## 🔧 Local Development

To run locally:
```bash
python3 -m http.server 8000
```
Then visit: http://localhost:8000/index-galaxy.html
